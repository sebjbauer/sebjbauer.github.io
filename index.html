Message Passing in Graph Neural Networks

This visualization demonstrates how message passing works in Graph Neural Networks. Colored particles show the flow of information between nodes, with each node's color influencing its neighbors through multiple layers of message passing.

Single Molecule Localization Microscopy (SMLM)

This visualization demonstrates how Single Molecule Localization Microscopy (SMLM) works. SMLM achieves super-resolution by exploiting the blinking behavior of fluorophores, allowing us to locate individual molecules with high precision.

Panel 1: True Structure

Shows the underlying structure with currently active fluorophores highlighted.

Panel 2: PSF Image

Displays how the microscope actually sees the active fluorophores.

Panel 3: Localizations

Shows the precisely determined positions of active fluorophores.

Panel 4: Accumulated

Builds up the super-resolution image over time.

Panel 5: Conventional

Shows the conventional microscope view.
