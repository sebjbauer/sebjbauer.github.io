Single Molecule Localization Microscopy (SMLM)

This visualization demonstrates how Single Molecule Localization Microscopy (SMLM) works. SMLM achieves super-resolution by exploiting the blinking behavior of fluorophores, allowing us to locate individual molecules with high precision.

Panel 1: True Structure

Shows the underlying structure with currently active fluorophores highlighted. Only a small subset of fluorophores is active at any time.

Panel 2: PSF Image

Displays how the microscope actually sees the active fluorophores, with each molecule appearing as a diffraction-limited spot.

Panel 3: Localizations

Shows the precisely determined positions of active fluorophores in the current frame.

Panel 4: Accumulated

Builds up the super-resolution image by accumulating all localizations over time.

Panel 5: Conventional

Shows how the sample would appear in a conventional fluorescence microscope.
